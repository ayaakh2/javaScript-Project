# Exam Tracker - Student Exam & Grade Tracking System

A complete web-based application that enables teachers to create and manage exams while allowing students to take exams, review their grades, and track their academic performance.

This project was developed as part of the **Orange Coding School** using HTML, CSS, Bootstrap, and Vanilla JavaScript.

---

# Project Overview

Exam Tracker is an educational platform designed to simplify exam management for teachers and improve the student learning experience.

The system provides two main roles:

- Teacher
- Student

Each role has its own dashboard with dedicated permissions and features.

---

# Features

## Teacher

- Secure login
- Teacher dashboard
- Add new students
- View and manage students
- Create exams
- Edit/Delete exams
- Activate or deactivate exams
- Support multiple question types:
  - Multiple Choice
  - True / False
  - Multiple Answers
  - Numeric Questions
- View students' grades
- Review submitted answers
- Track exam history

---

## Student

- Secure login
- Student dashboard
- View available exams
- Submit exams (one attempt only)
- View exam results
- Review:
  - Questions
  - Student answers
  - Correct answers
  - Mistakes
- View grades history
- Personal profile

---

- Home
